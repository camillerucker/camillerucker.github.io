3D Maze Game Java, LWJGL

Description

This 3D maze game was made from scratch in java using LWJGL. The player wanders in a randomly generated maze, and races the clock to find his/her way out. Power-ups scattered through out the maze give the players the ability to jump higher, run faster, and even grow in height

Technical Challenges

  • Checking for collisions with a wall in the maze in Constant time
  • Randomly generating the maze
  • Adding textures and skyboxing
