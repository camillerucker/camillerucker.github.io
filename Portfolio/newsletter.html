Exonian Email Newsletter Web, Java

Description

It is often hard to remeber when a new issue of The Exonian, Phillips Exeter's online newspaper, comes out. This project allows users to subsribe to an email list from The Exonian website. Whenever a new issue comes out, I send a nicely formatted HTML email with some of the issue's highlights and links to the articles. Generating a new newletter every week can be difficult, so I wrote a java program to generate it for me.
