IOT Sprinkler Web, Python, Arduino

More Pictures

Description

This IOT sprinkler system, made for the MIT media lab, incorporates a moisture sensor, and valve control over the whole yard. Never again will water be wasted after a rainstorm, when we are not there to turn off the sprinkler.

Technical

The front end website is connected to a firebase database, which saves the value of the sprinkler. On the copmputer, a python program gets the data from the database, and sends it via bluetooth to the sprinkler module. The sprinkler module receives the signal using its bluetooth receiver, and uses a relay to change the valve

The 2 step communication method creates a IOT framework. If an additional device were added to the system, instead of needing a costly wifi connection, all if would need is bluetooth.
