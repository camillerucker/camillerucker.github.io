Exeter Assembly TimeLine Java, Web Scraping

Description

Democracy of Sound, a club of which I am head hosted an assembly. Various musical groups sang together. The timeline of Exeter's history allowed the stuents to learn about Exeter's history, and the performers used the dates to remeber when they should enter the stage

Technical

A program parses the dates and events from this website on Exeter's website. Another java Swing program creates a GUI which cycles through all of the events. Before this program, the club's plan was to manually create a slideshow of the 485 events.
